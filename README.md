# Seme Logistics Website Project

## Overview
This project contains a modern, high-end website for **Seme Logistics**, fully implemented in Dutch.

### Features
- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Dark Theme**: Professional dark aesthetic with neon blue accents.
- **Fast Performance**: Built with pure HTML, CSS, and minimal JS.
- **Services Showcase**: Dedicated sections for all 5 requested services.
- **Contact Info**: Integrated phone and email details.

## Deployment to GitHub Pages
Since your GitHub is connected, you can deploy this site easily:

1.  **Push to GitHub**:
    -   Publish this repository to your GitHub account (using VS Code "Publish to GitHub" or command line).
2.  **Enable GitHub Pages**:
    -   Go to your repository **Settings** on GitHub.
    -   Click **Pages** in the left sidebar.
    -   Under **Source**, select `main` (or `master`) branch.
    -   Click **Save**.
3.  **Done!** Your site will be live at `https://[your-username].github.io/site-transport/`.

## Editing Content
-   **Text**: Edit `index.html` to change text.
-   **Styles**: Edit `style.css` to change colors or fonts.
-   **Images**: Replace images in the `assets/` folder.

## Viewing Locally
Open `index.html` in your browser to see the site immediately.
